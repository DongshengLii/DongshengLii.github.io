Peace Inside II（待续）

发表于 2017-03-01 | 分类于 当然我在扯淡 | | 阅读次数

其实我就是想写点什么了，继续在这里说出我想说的话。我越来越喜欢写东西，也许是因为这段时间我终于结束了自己阶段性的迷茫，在思考上有很大的收获，希望能用文字将它们梳理出来。期望这也能提升自己的说话水平吧？

四

大三下，似乎是大学生活结束前的最后一个重要的学期：保研的同学要准备参加夏令营，考研的同学要从这学期开始复习，而对于要出国的同学，这也是提升自己申请竞争力的最后一个学期。

在周一上完算法课回宿舍的路上，我感叹大家都在闷头做事的气氛挺好的，但这引发了同行同学的吐槽：大家都在埋头做自己的事情，却从来没有想过要一起共享信息等等互帮互助的想法。我想了想，确实，无论是从选择哪条去路，大家似乎都是形单影只，从来好像敌人一样相互提防着别人。很久以前我就注意到这件事，不过似乎已经习以为常了，这位同学提醒了我：世界本不应该是这个样子。

唉，但随后想想，这其实是大的文化环境铸就的人性，又怎么能凭借我们一己之力来改变呢？确实，我们可以振臂一呼，成立一个又一个保研、考研、出国交流群，可是谁加群的目的不是为了获取别人辛苦搜集来的资料呢，呵呵，从以往的很多事情的经验来判断，这不需要尝试也能够知晓其必然失败的结果。

既然大环境已经无法改变，我们就只能从个人层面寻求更好的解决方式：或许并不是大家都这样自顾自地发展，或许有人能够怀抱一颗相对公正的心态去和别人进行小范围的交流。不过这些，就因人而异了。

就像朴槿惠赠予芮成钢的题词：“人生在世，只求心安理得就好了”，无论别人是怎样选择，我只求自己能做到公正对待别人，至少在别人需要帮助的时候如果有能力就施以援手，坦荡对待别人，不去强求别人也要对我如何如何。心安理得，我认为目前就够了，毕竟，目前还有更重要的事去做。

五

今天去上了最优化的课程。

最优化对我来说是一门必修课，因为研究生要转IEOR方向以期获得相对来说高一点的大学排名。再在研究生阶段修一些计算机课程，在CS方面有所掌握，当然能去DataMining方向的IEOR是完美结果。

因为陈老师说想要拿一个高分会很难，所以课程上到第二节课，已经有一部分人离开。我印象深刻的是统计好友说了一句话：“这门课挺有用的，我觉得挺好的”，然后用脚投票，离开了教室。这让我想起来昨晚在宿舍走廊他们为寻求哪门课老师给分高所进行的交流。

可能对高成绩无欲无求，单凭给分高这一个理由并不能让我去选一门课。是的，成绩是很高，可是我又从这门课中学到了什么呢？或许只是在厌恶中又度过一个学期。之前是有这么一个说法：成绩不高，你连好的学校都去不了，你有什么资格谈能力呢？在现在的我看来，成绩已经只是一个数字，成绩所能代表的东西真的是寥寥无几，并且成绩高并不代表技能的水平就高，等到以后需要解决实际问题时，并不是说我把高分成绩单一拍，问题就自动解决的。相比较于成绩，我目前欠缺的更多的是能力，一种不至于教授邀请我到实验室做科研而我会心虚的能力。

写到这我不是要批判什么，而是想要说清楚自己现在的想法：就像打WOW，有人就喜欢刷任务，走制作方提前设计好的剧情并乐在其中，这当然无可厚非。而我目前，却想试试刷副本，这可能会付出很多却最终也看不到什么回报，不过过程中那种偶然掉落“好装备”的惊喜之感却是我格外珍视的。

六

今天吃饭路上遇到了班里成绩比较好的同学，他在准备保研。吃饭路上，我听到他说反正都是混日子……活着多么没意思……自杀算了等等一类的话，这和昨晚深夜好友与我说的话几乎完全一致。

或许我们这一类人都会经历这样一段时期，觉得自己比上不足，比下有余，似乎这辈子也看不到越过阶级门槛的希望，只是混日子，觉得人生顿时失去了意义。不过经过前一段时间的迷茫，我现在觉得自己已经走出这一时期：其实吧，人生本来就是分了不同的难度的，如果能通关进入更简单的难度固然很好，但是即使不能改变模式，能好好地玩下去，做一些自己觉得有意义的事情也是很棒的！当然，这是建立在能够依靠自己的独立意识分辨哪些是自己真正觉得有意义的基础之上的，没有这种意识，其实你也是活在别人给你写的剧本里，当然会觉得没意思。

哎，我很崇尚这样一句话，也希望我能够实现：如果非要将人生比作一场马拉松，那我就算爬也要一直爬到生命的尽头，在最后的时刻，用手重重地将重点线拍下，大喊一声“哈！我跑完了！”然后永远的休息。
