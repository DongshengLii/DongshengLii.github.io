简介

机器学习可以分为监督学习和无监督学习两种：这两者的区别在于监督学习中的数据是已经进行了标记（标签）的，明确地划分为不同的类，可以进行测试集的预测；而无监督学习则不需要对数据进行标注，着重研究数据本身的特性，不能进行预测。
