Catching Method for Solving Three Diagonal Equations

Posted on 2017-03-07 | In Numerical Calculations | Visitors

This semester I take the course of “Numerical Solution of Partial Differential Equation”, and I think it is REALLY FUN! So, I will share some of my course work and we can discuss these interesting problems together!

Today I am going to talk about a pretty easy problem and its algorithm: catching method for solving three diagonal equations and some special examples as an extension!

FYI, the fomula of my article is made by using the python package “OmniMarkup Preview”, and just change the setting “mathjax_enabled” into “true”.

Intro to the problem

So, as you can see, the equation $Ax = f$ is called a three diagonal equation if:

$
A{\rm{ = }}\left( {\begin{array}{*{20}{c}}
{a_1}&{c_1}&{}&{}&{}\\
{b_2}&{a_2}&{c_2}&{}&{}\\
{}& \ldots & \ldots & \ldots &{}\\
{}&{}& \ldots &{a_{n - 1}}&{c_{n - 1}}\\
{}&{}&{}&{b_n}&{a_n}
\end{array}} \right)
$,$
f{\rm{ = }}\left( {\begin{array}{*{20}{c}}
{f_1}\\
{f_2}\\
\ldots \\
{f_{n - 1}}\\
{f_n}
\end{array}} \right)
$
and there is a pretty clear and simple algorithm called “catching method”, which I think its exciting to derive, so let`s do it!

The principle of the solution (ordinary)

Let the coefficient matrix A of the n-th linear equations ${\rm{Ax = f}}$ be a nonsingular tridiagonal matrix:

$$A{\rm{ = }}\left( {\begin{array}{*{20}{c}}
{a_1}&{c_1}&{}&{}&{}\\
{b_2}&{a_2}&{c_2}&{}&{}\\
{}& \ldots & \ldots & \ldots &{}\\
{}&{}& \ldots &{a_{n - 1}}&{c_{n - 1}}\\
{}&{}&{}&{b_n}&{a_n}
\end{array}} \right)$$

“LU decomposite” this matrix:
$$A = LU = \left( {\begin{array}{*{20}{c}}
{l_1}&{}&{}&{}&{}\\
{m_2}&{l_2}&{}&{}&{}\\
{}& \ldots & \ldots &{}&{}\\
{}&{}& \ldots &{l_{n - 1}}&{}\\
{}&{}&{}&{m_n}&{l_n}
\end{array}} \right) \times \left( {\begin{array}{*{20}{c}}
1&{u_1}&{}&{}&{}\\
{}&1&{u_2}&{}&{}\\
{}&{}& \ldots & \ldots &{}\\
{}&{}&{}&1&{u_{n - 1}}\\
{}&{}&{}&{}&1
\end{array}} \right)$$

Where L is the lower triangular matrix and U is the triangular matrix on the unit. Using the matrix multiplication method, we can get:

$$\begin{array}{l}
{l_1} = {a_1}\\
{l_1}{u_1} = {c_1}\\
{u_1} = {c_1}/{l_1}\\
{m_i} = {b_i}(2 \le i \le n)\\
{a_{i + 1}} = {m_{i + 1}}{u_i} + {l_{i + 1}}(1 \le i \le n - 1)\\
{c_i} = {l_i}{u_i}\\
{u_i} = {c_i}/{l_i}\\
{l_{i + 1}} = {a_{i + 1}} - {b_{i + 1}}{u_i}
\end{array}$$

L and U can be obtained from the above types. Let $y = Ux$, then we can get:

$$\begin{array}{l}
Ax = LUx = Ly = f\\
Ly = f
\end{array}$$

we already know L and d, so we can get y:

$$\left( {\begin{array}{*{20}{c}}
{l_1}&{}&{}&{}&{}\\
{m_2}&{l_2}&{}&{}&{}\\
{}& \ldots & \ldots &{}&{}\\
{}&{}& \ldots &{l_{n - 1}}&{}\\
{}&{}&{}&{m_n}&{l_n}
\end{array}} \right) \times \left( {\begin{array}{*{20}{c}}
{y_1}\\
{y_2}\\
\ldots \\
{y_{n - 1}}\\
{y_n}
\end{array}} \right) = \left( {\begin{array}{*{20}{c}}
{f_1}\\
{f_2}\\
\ldots \\
{f_{n - 1}}\\
{f_n}
\end{array}} \right)$$

so we can get the expression of y:

$$\begin{array}{l}
{y_1} = {f_1}/{l_1}(i = 2,3, \ldots ,n)\\
{m_i}{y_{i - 1}} + {l_i}{\rm{y}_i}{\rm{ = }}{f_i}\\
{y_i} = \left( {f_i} - {m_i}{y_{i - 1}} \right)/{l_i}
\end{array}$$
for $y = Ux$, we can get:
$$\left( {\begin{array}{*{20}{c}}
1&{u_1}&{}&{}&{}\\
{}&1&{u_2}&{}&{}\\
{}&{}& \ldots & \ldots &{}\\
{}&{}&{}&1&{u_{n - 1}}\\
{}&{}&{}&{}&1
\end{array}} \right) \times \left( {\begin{array}{*{20}{c}}
{x_1}\\
{x_2}\\
\ldots \\
{x_{n - 1}}\\
{x_n}
\end{array}} \right) = \left( {\begin{array}{*{20}{c}}
{y_1}\\
{y_2}\\
\ldots \\
{y_{n - 1}}\\
{y_n}
\end{array}} \right)$$

so x is as follows:
$$\begin{array}{l}
{x_n} = {y_n}\left( {i = n - 1,n - 2, \ldots ,1} \right)\\
{x_i} = {y_i} - {u_i}{x_{i + 1}}
\end{array}$$

“Advanced Play!”

Prof.Ji gave us a taxing question: if A is a matrix like below,$p,q \ne 0$, what should I do to solve this problem?
$$A{\rm{ = }}\left( {\begin{array}{*{20}{c}}
{a_1}&{c_1}&{}&{}&q\\
{b_2}&{a_2}&{c_2}&{}&{}\\
{}& \ldots & \ldots & \ldots &{}\\
{}&{}& \ldots &{a_{n - 1}}&{c_{n - 1}}\\
p&{}&{}&{b_n}&{a_n}
\end{array}} \right)$$

The Solution of”Advanced Play!”

Let the coefficient matrix A of the n-th linear equations ${\rm{Ax = f}}$ be a nonsingular tridiagonal matrix,$p,q \ne 0$:
$$A{\rm{ = }}\left( {\begin{array}{*{20}{c}}
{a_1}&{c_1}&{}&{}&q\\
{b_2}&{a_2}&{c_2}&{}&{}\\
{}& \ldots & \ldots & \ldots &{}\\
{}&{}& \ldots &{a_{n - 1}}&{c_{n - 1}}\\
p&{}&{}&{b_n}&{a_n}
\end{array}} \right)$$

“LU decomposite” this matrix:

$$A = LU = \left( {\begin{array}{*{20}{c}}
{l_1}&{}&{}&{}&{}\\
{m_2}&{l_2}&{}&{}&{}\\
{}& \ldots & \ldots &{}&{}\\
{}&{}& \ldots &{l_{n - 1}}&{}\\
{\alpha _1}& \ldots &{\alpha _{n - 2}}&{m_n}&{l_n}
\end{array}} \right) \times \left( {\begin{array}{*{20}{c}}
1&{u_1}&{}&{}&{\beta _1}\\
{}&1&{u_2}&{}& \vdots \\
{}&{}& \ldots & \ldots &{\beta _{n - 2}}\\
{}&{}&{}&1&{u_{n - 1}}\\
{}&{}&{}&{}&1
\end{array}} \right)$$

Where L is the lower triangular matrix and U is the triangular matrix on the unit. Using the matrix multiplication method, we can get:
$$\begin{array}{l}
{l_1} = {a_1}\\
{u_i} = {c_i}/{l_i},\left( 1 \le i \le n - 2 \right)\\
{u_{n - 1}} = \left( {c_{n - 1}} - {m_{n - 2}}{\beta _{n - 2}} \right)/{l_{n - 1}}\\
{l_{i + 1}} = {a_{i + 1}} - {m_i}{u_i},\left( 1 \le i \le n - 2 \right)\\
{l_n} = {a_n} - \sum\limits_{i = 1}^{n - 2} {\alpha _i}{\beta _i} - {m_{n - 1}}{u_{n - 1}}\\
{m_i} = {b_i},\left( 1 \le i \le n - 2 \right)\\
{m_{n - 1}} = {b_{n - 1}} - {\alpha _{n - 2}}{u_{n - 2}}\\
{\beta _1} = q/{l_1}\\
{\beta _{i + 1}} = \left( {m_i}{\beta _i} \right)/{l_{i + 1}},\left( 1 \le i \le n - 3 \right)\\
{\alpha _1} = p\\
{\alpha _{i + 1}} = - {\alpha _i}{u_i}
\end{array}$$
L and U can be obtained from the above types.
Let $y = Ux$, then we can get：
$$\begin{array}{l}
Ax = LUx = Ly = f\\
Ly = f
\end{array}$$
we already know L and d, so we can get y:
$$\left( {\begin{array}{*{20}{c}}
{l_1}&{}&{}&{}&{}\\
{m_2}&{l_2}&{}&{}&{}\\
{}& \ldots & \ldots &{}&{}\\
{}&{}& \ldots &{l_{n - 1}}&{}\\
{\alpha _1}& \ldots &{\alpha _{n - 2}}&{m_n}&{l_n}
\end{array}} \right) \times \left( {\begin{array}{*{20}{c}}
{y_1}\\
{y_2}\\
\ldots \\
{y_{n - 1}}\\
{y_n}
\end{array}} \right) = \left( {\begin{array}{*{20}{c}}
{f_1}\\
{f_2}\\
\ldots \\
{f_{n - 1}}\\
{f_n}
\end{array}} \right)$$

so we can get the expression of y:
$$\begin{array}{*{20}{l}}
{y_1} = {f_1}/{l_1}\\
{m_i}{y_{i - 1}} + {l_i}{\rm{y}_i}{\rm{ = }}{f_i}\\
\begin{array}{l}
{y_i} = \left( {f_i} - {m_i}{y_{i - 1}} \right)/{l_i},(i = 2,3, \ldots ,n - 1)\\
{y_n} = \left( {f_n} - \sum\limits_{i = 1}^{n - 2} {\alpha _i}{y_i} - {m_{n - 1}}{y_{n - 1}} \right)/{l_n}
\end{array}
\end{array}$$
for $y = Ux$, we can get:
$$\left( {\begin{array}{*{20}{c}}
1&{u_1}&{}&{}&{\beta _1}\\
{}&1&{u_2}&{}& \vdots \\
{}&{}& \ldots & \ldots &{\beta _{n - 2}}\\
{}&{}&{}&1&{u_{n - 1}}\\
{}&{}&{}&{}&1
\end{array}} \right) \times \left( {\begin{array}{*{20}{c}}
{x_1}\\
{x_2}\\
\ldots \\
{x_{n - 1}}\\
{x_n}
\end{array}} \right) = \left( {\begin{array}{*{20}{c}}
{y_1}\\
{y_2}\\
\ldots \\
{y_{n - 1}}\\
{y_n}
\end{array}} \right)$$

so x is as follows:
$$\begin{array}{l}
\begin{array}{*{20}{l}}
\begin{array}{l}
{x_n} = {y_n}\\
{x_{n - 1}} = {y_{n - 1}} - {x_n}{u_{n - 1}}
\end{array}\\
{x_{n - 1 - i} = {y_{n - 1 - i}} - {u_{n - 1 - i}}{x_{n - i}} - {\beta _{n - 1 - i}}{x_n}\left( {i = n - 2, \ldots ,1} \right)}
\end{array}\\
\end{array}$$
