卑鄙是卑鄙者的通行证

发表于 2017-02-25 | 分类于 当然我在扯淡 | | 阅读次数
LIFE VERSION 0.1

一

翻来覆去睡不着。

回想家族中长辈的历史，现在感觉似乎能够体会到一点什么。

A生于战争年代。父亲是私塾老师，但在A幼年之时就因病去世，只剩母子二人相依为命。A从小放牛为生，曾藏在牛肚子下面与日本鬼子擦肩而过而活命。后来新中国成立，A凭借自己努力从一个穷学生到高级工程师，再到某国有陶瓷厂厂长，当时正处国有企业的巅峰时期，A的等级与市长平起平坐。出国考察，国内指导，都不在话下。后来A也顺利成家，娶了家境很是不错的大户人家的小姐。再后来，国有企业制度改革的前夕，A将妻子、子女也安排在了他所在的厂，子女也成家立业，一切岁月静好。后来，国企改革，效益不如以前。没有铁饭碗的人们纷纷“下海”，而A与其家人们当然不愿意再白手起家，依旧对老厂不离不弃，直到破产，期间有人聘请A一家去某大学当教授也婉然谢绝。如今，A的子女当然都不好过。

…………

只是觉得后怕：以前只是觉得人是环境的产物，而现在或许应该改成“人是家庭环境的产物”。至少年轻的时候的选择总是会与家庭的教育和环境有渊源。

以前总会有一种自己是天之骄子的傲气，而现在，只觉得自己与别人的差距越拉越大。大学以来，总喜欢以一个理由来安慰自己“在中国，高考之前的人生其实大家都是一样的、相当于浪费的度过”。而现在才猛然醒悟，其实有相当一部分人是早就有了真正是自己的想法的，而我呢？高中写作文用个素材还要靠“XX素材”文章的评论来撑内容。

二

现在，我们差距还在不停地扩大，这区别的源头有一点是在于有没有想法。易经上说，人的一生大概会有七次改变命运的机会：前一两次因为年纪太小，常常无法把握；后一两次因为年纪太大也不能强求。只有二、三十岁的两三次机会，是完全握在自己手心里的。最有可能是这两次机会的，也就是研究生和工作吧？

初、高中多年的好兄弟，因为高考失利而去了山大学医。且不说崇尚当医生和考公务员是山东人特有的“地域特色”，寒假回家与他交流时，得知他已经着手准备保研本校，不出意外的话最终能到山东境内的大医院开始“往上爬”。听了之后我感到有些惋惜：从心底里讲，我知道他心里不是真喜欢学医，学医是家中长辈的心愿。而我确实认为，就算是学医，以他的智商和努力，特值得去国内顶尖的学校。可是在他周围皆是各方面比不上他的人，为保研争得头破血流，他又如何能容易地意识到有更好的平台在等着他呢？我为他感到不甘。

不过这么说总让我对自己有一种“何不食肉糜”的愚蠢，那么，对于我自己呢？

现在我似乎手上握着这样的一次机会：出国求学。可我现在站在好几扇门前，手中的“钥匙”却只能开其中的一扇——没有回头路。前一阵子几乎陷入了全面的纠结：中介哪家好？选学校还是选专业？专业哪个适合我？暑期科研怎么找？套磁怎么套？……然而现在内心已经回归平静：从来就没有什么可以纠结的东西，我要的人生从来就不是一股“洪流”裹挟我前进，如果现在还身陷“洪流”无法自拔，那最终的去向是哪里我还不知道吗？其实到现在我还是在根据别人的建议选择专业，其实和高考志愿的XJB填是一样的，最终的结果满不满意别人反正也不敢保证的咯。而现在，我似乎已经有了答案。

三

开学之后，各位同学都陆续回来，与之二三交谈，得出以下几个感悟：

绝大多数人只愿意让你共富贵，不可共贫贱

这句话的意思是：你有好事发生，他们希望让你分享；你遭遇不幸，他们却毫不在意。反过来，他们有好事不会找你，遇到困难却会要求你施以援手。

本来并不是一个很冷漠的人，但是经历了一系列的事情之后，现在的我对周围环境是非常失望。可能道理就如同“你是钉子，那你看周围的一切都是锤头”一样吧？如果顺风顺水当然不会在意别人是否真心，但当真的需要别人援助之时就会体味人情冷暖，

请勿交浅言深

这道理结合上面就很容易得到了：别人都不是真心希望我好，我为什么还要把自己内心暴露出来呢？有的时候我总会忍不住把自己的一些想法、困惑与并不值得的人交流，现在看来除了被利用还得不到半句感谢，还是太天真了呢。

以后的一段时间内我估计就在博客上倾吐自己的心声吧，免得再发生类似不好的事情。

不要愚蠢的高尚

这是计算代数课程带给我的感悟：何青自己都不尊敬自己的课程，那我还有什么理由去尊敬他呢？以后也是这样，不要因为自己心里觉得高尚就去吃亏，其实到最后发现“卑鄙”的人混的好，心里会更难受。

“卑鄙是卑鄙者的通行证，高尚是高尚者的墓志铭”，高中的时候只是将这首诗进行了天真的理解，可现实却是：卑鄙者凭通行证在康庄大道上畅行无阻，而高尚者却只能将高尚宣扬在荒山野岭的乱坟岗。

想想确实难过，但是这就是社会，呵呵。我变了，也是从现在开始彻底转变了，谢谢您，何青老师。

LIFE VERSION 0.1

活了这么多年，我对世界的态度和对未来的规划经历了数次变化，说实话我庆幸于这种改变，因此我想对现在的规划做一次记录，或许以后会有改变，就暂且叫它LIFE VERSION 0.1吧。

考虑到家里的经济状况，目前我能想到的只有三种选择：1.正常申请master，走CSC奖学金；2.走免学费国家（如德国）项目；3.有奖学金的master或者phd

研究生修读计算机方向（或者金融相关方向但是也要重视计算机），不读phd直接去科技公司或者金融公司就业。在国内的大城市（北上港）落户。

目前只能想到这，再往后想估计只能算YY了吧哈哈~

# LIFE VERSION # 近期感悟
