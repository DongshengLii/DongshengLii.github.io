发表于 2017-02-14 | 分类于 当然我在扯淡 | | 阅读次数

开篇

没想到搭建一个博客原来这么简单，从决定转而使用静态语言搭建到基本完成上线用了不到24小时。虽然目前还有很多自己觉得不完美的地方，但是基本已经足够我使用啦！当然要感谢其他小伙伴对我问题的及时解答，我决定近期把我搭建这个博客的全过程放上来，希望更多的小伙伴能有自己的博客！（下面是早就写好的文字，就当作是我博客之路的Hello World吧~）

为什么想要建立这个博客

写博客这个想法由来已久，也尝试已久。

首先，我这个人很喜欢胡思乱想，但是始终没有形成一种良好的思维方式，导致目前我的思维其实是杂乱无章的。长久以来我一直寻求一种能够理清自己思路的方式，尝试了一些方法，至今都没能坚持下来，非常可惜。目前思考的深度在我看来一直非常粗浅，在与朋友们交谈的时候总感觉自己内蕴不足（这也是目前我不太出来与大家一起交流的原因），这让我非常懊恼。

其次，有些事情、有些想法如果不能整理出来，也会一直在脑海中挥之不去，阻碍我的思考。

最后，开博客是希望大家能够看到。我实在是不喜欢发朋友圈这种简短的文字，而“或许”会有朋友想要了解我的生活思维状态；此外，这也逼迫了我定期写作，同时也欢迎朋友们一起交流。

我对这个网站的期望不仅仅是一个只倾诉个人情感、想法的博客，更是一个技术网站。我会将自己认为有用的知识和读后感分享给大家，同时也作为一种知识掌握程度的检查，如果我不能给大家讲明白，那说明我对这一知识也不是全面的掌握。
