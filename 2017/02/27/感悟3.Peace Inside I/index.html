Peace Inside I

发表于 2017-02-27 | 分类于 当然我在扯淡 | | 阅读次数

最近经历了诸多事情，内心斗争极度频繁但最终又回归平静。不敢贸然使用Inner Peace这一高端词汇，但我现在至少是做到了Peace Inside了。

一

现在看来已经是很可笑的：前不久我还为一门自己付出较多精力的专选课程老师给分不高而生了很大一通气。过了一阵想想，不就是一门课吗？自己也只是看似努力，而自己的能力却没有得到实质性的提高。现在看自己的CV，如果我是老板的话，我当然也不会给我机会，更何况已是阅人无数的教授呢？

所以提升自己的能力还是燃眉之急：这学期本来希望一门课不选直接准备GT，然而，仔细想想觉得，无论从什么角度考量，整个学期的时间只用来准备标准化英语考试无疑是一种浪费。所以还是忙一些，充实一些：今天早上去听了信科数据挖掘的课程，教室里的大多数学生都昏昏欲睡而我却感觉热血沸腾————确实，很久没有听自己想听的课程了。课间有那么一刹那询问自己是否后悔选择了数学，但是答案很明确是“否”：人生其实就是一次次赌博，在填报志愿的时候，面对让自己一脸茫然的各种专业，“吃低保”是最稳的选择，信科选择看似现在收益极大，可别忘了自己当初可是脑海中一直高亮着“21世纪时生物的世纪”这一口号的。就算不选择数学，第二选择估计也不是信科。

所以，现在选了一大堆计算机的专业必修课，为的是以后能给自己留一条转CS专业的后路。何况，不像辅修，跨专业修课又不花钱也没有任何限制，何乐而不为？

二

今晚偶然遇到了微信中素未谋面的好友，虽然早就从别人口中听说此人所作所为，不过亲眼所见其不文明举止还是能引起我的极大厌恶。果断删之，管她是要去哥大统计还是哈佛统计。

不过，闭馆的路上，脑海中回响起一同旁听算法课的大神同学的人生哲学：不要管别人如何，做好自己。确实，我有时候总是会内心偷偷地基于自己所见对身边的人做出评价，这确实不是一个好习惯。不过，至少要保留一个底限：不能恶心到我。

可能是自己内心还有一股早就不应该有的傲气，每当想起自己身边时不时地会出现一两个“败兴之人”，就为自己感到不甘，所以要更加努力，拒绝将世界拱手让给自己所鄙夷之人。

三

虽然我不相信命运一说，但是其实一个人的未来走向和其家境，观念，视野等等都不无关系，“可怜之人必有可恨之处”也是这个道理。不过，我认为计算机发展带来的大数据时代可能是一个“翻身”的好机会：信息从某种程度上来讲将不在完全把握在有权势财富的阶级手中，如果有技术与知识，或许信息和其所带来的财富能为你所拥有。这句话先放在这，或许哪一天自己回头再看，能触发许多感慨吧！

其实，长者曾经说过，只有琢磨透历史的人才能把握住时代的机遇。我认为有道理，在日后的学习生活中，也应该注重看一些文史哲的书，或许能带来不一样的视野与想法。
